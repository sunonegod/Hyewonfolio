안녕하세요 저는

새로운 길을 서슴없이
나아갈 수 있는 개발자입니다.

고등학생 시절에는 미술대학교에 진학하기 위해 미술 공부를 했습니다. 계원예술대학교에 진학하여 1학년 때
기획, 디자인, 프로그래밍 세 분야를 다양하게 배우면서 기획에 대한 이해와 디자인에 대한 기초 지식을 토대로
웹사이트를 제작했습니다. 기획과 디자인에서 마치는 것이 아닌 실제로 웹사이트를 제가 쓴 코드로 구현해
나간다는 과정에서 큰 성취감을 느꼈습니다. 디자인만을 생각해 왔던 저에게 새로운 도전을
시작하게 해준 코딩의 매력에 매료되어 2학년 세부전공으로 프로그래밍을 전공하였습니다.

발전 가능성이
있는 개발자입니다.

디자이너 친구들과 팀을 이뤄 하나의 애플리케이션을 제작하는 프로젝트 스튜디오 수업을 들었습니다.
저는 팀 내에서 제가 다룰 수 있는 프로그래밍언어를 사용하여 팀이 제작한 앱 서비스를 소개하는 웹사이트를
만드는 역할을 맡았습니다. 하지만 1학년 때 배운 html과 css 언어만으로는 웹사이트를 만드는 것은 인터랙션과
기능적인 부분이 부족하다고 느꼈습니다. 검색 능력을 활용하여 다양한 자바스크립트 기능을 찾고,
적용하며 스스로 한 단계 더 성장하는 계기를 갖게 되었습니다.
그 결과 커뮤니케이션 디자인 국제 공모전에서 출품한 작품으로 좋은 성과를 이뤘습니다.

팀 작업의 매력을
아는 개발자입니다.

대학교 2학년 1학기 계절학기로 전주대학교에서 주최한 ‘소셜리빙랩실감미디어PBL’
수업을 들었습니다. 팀을 이뤄 기후 환경에 대한 주제로 매드맵퍼 프로그램을 활용하여
미디어파사드 작업을 진행하였습니다. 기업이 배출하는 쓰레기의 양과 가정에서 버리는
쓰레기 의 양의 차이를 객체가 랜덤으로 나오는 모습으로 보여주며 기업에서도 기후 환경에
대한 경각심을 일으켜야 한다는 메세지를 주고자 하였습니다. 그리고 기업 뿐만 아니라
우리 일반인들도 기후 환경에 대한 목소리를 높여야 한다는 의미를 담고자 구글 API 코드를
사용하여 화면에 나오는 글을 읽으면 쓰레기가 사라지는 기능을 구현했습니다.
이런 활동을 통해 파이썬을 경험하고, 구글 API 서비스를 알게되었습니다.
팀 내에서 일러스트 툴을 다룰 줄 아는 저는 주로 소스로 사용할 일러스트를 작업하는
역할을 맡고, 프로젝트에 어울릴 코드들을 찾아주는 역할을 맡았습니다.
서로의 역량에 맞는 역할을 맡아 하나의 작품을 완성해 간다는 것이 공동체를
경험하게 해 주었습니다. 관객들 앞에서 팀의 작풍을 선보일 때의 짜릿함과
뿌듯함이 팀 작업의 매력을 한 층 더 느끼게 해 주었습니다.
